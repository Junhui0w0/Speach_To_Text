# Speach To Text

**2025-02-25**  
부모님 택배 사업에 도움이 될 수 있는 음성인식 프로그램  
GUI를 통해 필요한 데이터를 입력 가능  

[❗] 음성인식 도중에 버튼을 클릭하면 음성인식이 여러번 수행됨  
[❗] '주소' 입력 시 지번은 넘패드를 구현해 입력할 수 있도록 추가  
[❗] '연락처' 입력 시 넘패드를 구현해 입력할 수 있도록 추가  
[✔] '결제방식' 라디오버튼 배치 수정 (2025-02-26)
